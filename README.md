# Express JS + CURD + Template Engine
Backend with expressJS. Implemented CRUD operations. Tried template engine. 

## Concept covered
* Baisc syntax of a web server. 
* Basic route handling. 
* Express middleware
* Static folder. 
* Rendering template. 

## How to run? 
* `npm install`
* `npm start`
* Open `http://localhost:3000` in the browser.
